Sentinel — генератор конфигурации

Статический клиент (vanilla JS, без сервера и сети — работает offline / air-gapped). Собирает валидный RunConfig YAML + env-блок + команду запуска из проверенной поверхности конфигурации (brain/runconfig.py · cmd/agentctl · brain/llm.py). Модели и эндпоинты — в LOCAL_MODELS.md. ADR-031 фаза-1. Локально / air-gapped (часть Docker-бандла): docker compose --profile webui up → открой http://localhost:8088/setup/.
