Формула

VRAM ≈ params·bytes(quant) + KV + overhead; KV = 2·n_layers·kv_dim·ctx·bytes_per_elt / 1e9. Оценки ±10–15%. Для нагрузки Sentinel (вход ≤2000 tok) KV пренебрежимо. Сверяйте ollama show / llama.cpp --info.
